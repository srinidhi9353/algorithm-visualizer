### Port already in use?
```bash
# Kill the process or use a different port
npm run dev -- --port 3000
```

### Dependencies issues?
```bash
# Clear node_modules and reinstall
rm -rf node_modules package-lock.json
npm install
```

### Build errors?
```bash
# Check Node version (should be 18+)
node --version

# Update dependencies
npm update
```
